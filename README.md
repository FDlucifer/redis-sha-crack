redis-sha-crack
==========

Simple, distributed sha1 password cracking using redis 2.6 instances.

# Install Dependencies

``` npm install . ```

# Usage

``` node ./redis-sha-crack.js -w wordlist.txt -s shalist.txt 127.0.0.1 host2.example.com:5555 ... ```
